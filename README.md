# IA-na-Detec-o-de-Intrus-es-Otimiza-o-via-PCA
Estudo comparativo de Machine Learning aplicado à Detecção de Intrusões em Rede (NIDS) utilizando o dataset UNSW-NB15. O projeto foca na otimização computacional e redução de dimensionalidade através de PCA (20 componentes principais com 93.60% de variância explicada).
